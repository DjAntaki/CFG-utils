
Dependencies :
    nltk
    

Little script for quick use of nltk.grammar.CFG made for Pierre McKenzie.
The file cfg_prompt.py contains two prompts to create CFG grammars. 
The file cfg_utils.py contains functions to test if a grammar accepts a words, to
list accepted words under a given length, etc. 

To see the functions associated with nltk.grammar.CFG :
http://www.nltk.org/api/nltk.html#nltk.grammar.CFG
