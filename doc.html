CFG-utils¶

This script contains some functions to quickly use nltk.grammar.CFG to create and test context-free grammars.

Developped with python 2.7. by Vincent Antaki. Report bugs on the github page : https://github.com/DjAntaki/CFG-utils

If you want to use it with python >=3, use 2to3 (https://docs.python.org/2.7/library/2to3.html) to convert or change manually the call to function “raw_input” to “input”.

To see the functions associated with nltk.grammar.CFG : http://www.nltk.org/api/nltk.html#nltk.grammar.CFG

Documentation¶

cfg_utils._get_terminal_symbols(cfg)¶

Returns a set of all the terminal symbols used in a nltk context-free grammar.

cfg_utils._recognizes(parser, word)¶

Returns True if the CFG accepts that word and False if it doesnt.

parser : a nltk.parser instance for a cfg. word : a list of tokens

cfg_utils._rules_input_prompt(nonterminals, terminals)¶

Given a list of nonterminals and a list of terminals, this function creates a list of rules (aka grammar.Production instances) and returns it. There must be a whitespace between all terminals and nonterminals symbols. Extra whitespaces are ignored/conside.

cfg_utils.accepted_length(cfg, x)¶

Returns a list of every accepted word of a context-free grammar with a specific length

cfg_utils.accepted_under(cfg, length)¶

Returns a list of every accepted word of a context-free grammar under a given length. cfg : a nltk.grammar.CFG instance.

cfg_utils.language_equals_under(cfg1, cfg2, length)¶

Compares all accepted words under a given length accepted by context-free grammar 1 and context-free grammar 2.

cfg1 & cfg2 : nltk.grammar.CFG instances :return True if the they accept the same words, False if they do not.

cfg_utils.load_grammar(path)¶

Loads a nltk.CFG instance previously saved in binary with pickle at given path.

cfg_utils.quickprompt()¶

Interactive prompt to create a context-free grammar.

This prompt assumes that the non-terminal symbols is the uppercase alphabet and that the terminal symbols is lowercase alphabet. You only input rules and starting state.

cfg_utils.recognizes(cfg, word)¶

cfg : a nltk.grammar.CFG instance word : a string with tokens separated with spaces.

A parser is created at every call of this function.

cfg_utils.recognizesAll(cfg, words)¶

Returns a list of boolean values corresponding to [recognizes(cfg,w) for w in words]. cfg : a nltk.grammar.CFG instance words must be a list of string with tokens separated with spaces.
